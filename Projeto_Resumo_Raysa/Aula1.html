Aula 1


Objetivo: nesta aula daremos uma introdução ao html e como utilizar a plataforma Visul Studio Code(VS).



O que é HTML?

HTML é uma abreviação de Hypertext Markup Language, ou seja, Linguagem de Marcação de Hipertexto. Resumindo, o HTML é uma linguagem usada para a publicação de conteúdo (texto, imagens, vídeos, áudio etc.) na web.


Iniciando a criação do código:

Primeiramente, é necessário criar uma pasta onde serão salvos todos os arquivos que serão feitos. Para adicioná-la no VS, batar clicar em File, Open Folder e depois selecionar a pasta desejada dando enter depois. Uma vez que já temos a pasta, podemos manter uma organização criando pastas dentro dela, como por para imagens. O próximo passo é criar uma página html onde estará o nosso código principal do site que será criado, geralmente se chama "index.html".


Assim que já tivermos a página html criada no VS, podemos clicar em cima dela para abri-la e digitar "!" seguido de enter, é possível observar que irão aparecer alguns códigos de html 5 já prontos para a página. Nestes códigos contém algumas tags que explicaremos mais a frente. Faremos as seguintes modificações:

  1. Na tag "html lang='en'", mudar "en" para "pt-br", que informa a língua do código que no caso é potuguês;
  2. Na tag "title" coloque o título desejado para sua guia.



Entendendo os comandos e tags do html 5

São utilizados para informar ao navegador que tipo de estrutura será utilizada, como título, parágrafos, imagens, links e etc. Seguem alguns:

  • DOCTYPE: Define o tipo de documento, é aqui que o navegador entende o tipo de documento e como ele deve interpretar as tags nele contidas;

  • HTML: Define um documento html;

  • body: Define o corpo da página;

  • Head: Define uma informação sobre o documento;

  • title: Define o título do documento;

  • h1 até h6: Define do cabeçalho 1 até o cabeçalho 6;

  • p: Define um parágrafo;

  • strong: Define um texto em negrito;

  • em: Define um texto em itálico;

  • u: Define um texto sublinhado;

  • strike: Define um texto riscado;

  • hr: Define uma regra horizontal. (Tem o mesmo nível que um parágrafo, mas também é utilizado para fazer separações e quebras de linha);

  • img: Define uma imagem;

  • ul: Define uma lista desordenada;

  • ol: Define uma lista ordenada;

  • li: Define os itens da lista;

  • br: Define uma quebra de linha.

Inserindo imagens


Exite três maneiras que podemos inserir imagens. Observando o código abaixo, no primeiro comando img é a forma que se insere quando a imagem está na mesma pasta que o código html. No segundo comando, é o modo que podemos inserir através do link da imagem. No terceiro comando é utilizado um caminho, ou seja, se informa a pasta onde está contida a imagem, uma vez que ela está em pasta diferente do código html.

"src" é onde inserirmos o nome e (caso haja) caminho da imagem, "width" é a largura, "alt" a descrição e "height" a altura. Esses últimos destalhes são opcinais.



Conforme for salvando código aos poucos e queira ver como está ficando a página web há duas opções. A primeira é abrir o seu index normalmente através da sua pasta, assim cada vez que salvar o código será necessário atualizar a página para visualizar o novo conteúdo. A segunda é clicar com o botão direito em cima do arquivo no VS e depois ir em "Open With Live Server", assim a página ficará online e atualizará automaticamente.
